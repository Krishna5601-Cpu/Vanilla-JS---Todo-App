# ✅ Vanilla JS Todo App

A clean, lightweight Todo application built with pure **HTML**, **CSS**, and **Vanilla JavaScript** — no frameworks, no dependencies.

---

## 🚀 Live Demo

> Open `index.html` directly in your browser — no build step required.

---

## 📸 Features

- ➕ **Add tasks** — quickly create new todos via input field
- ✔️ **Mark as complete** — toggle tasks between done and pending
- 🗑️ **Delete tasks** — remove individual todos
- 💾 **Local Storage** — todos persist across browser sessions
- 📱 **Responsive design** — works on desktop and mobile

---

## 🗂️ Project Structure

```
Vanilla-JS---Todo-App/
├── index.html       # Main HTML structure
├── style.css        # Styling and layout
└── script.js        # App logic (add, complete, delete tasks)
```

## 🛠️ Getting Started

### Prerequisites

All you need is a modern web browser. No Node.js, npm, or build tools required.

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Krishna5601-Cpu/Vanilla-JS---Todo-App.git
   ```

2. **Navigate into the project folder**

   ```bash
   cd Vanilla-JS---Todo-App
   ```

3. **Open in your browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` in your file explorer.

## 🧠 How It Works

| Action                                         | Description                         |
| ---------------------------------------------- | ----------------------------------- |
| Type a task & press **Enter** or click **Add** | Creates a new todo item             |
| Click the **checkbox**                         | Marks a task as complete/incomplete |
| Click the **delete (🗑️) button**               | Removes the task permanently        |
| **Refresh the page**                           | Tasks reload from Local Storage     |
